# 설비 수주 자동응답 매크로 v1.4

카카오톡 PC 오픈채팅방을 실시간 감시하여, 지정 조건(지역·작업명·수수료)에 맞는 게시글이 올라오면 자동으로 응답 문구를 전송하는 Windows 전용 OCR 자동응답 매크로입니다.

v1.4 변경 사항:
- OCR 공백 노이즈가 있어도 지역/작업/수수료 매칭 강화
- OCR 최근 인식 / 후보 로그 추가
- 로그에 분:초 표기 및 조건충족 빨간색 표시
- 앱 초기 위치를 화면 중앙보다 약간 왼쪽으로 조정

## 요구사항

- **OS**: Windows 10 / 11
- **Python**: 3.8+ (설치 시 'Add Python to PATH' 체크 필수)
- **Tesseract OCR**: [다운로드](https://github.com/UB-Mannheim/tesseract/wiki) 후 설치
  - 설치 경로: `C:\Program Files\Tesseract-OCR`
  - **한국어 언어팩** 설치 필수 (`kor.traineddata`)

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python main.py
```

## 사용 방법

1. 카카오톡 PC 버전에서 감시할 오픈채팅방을 **화면에 열어 둡니다.**
2. 채팅방 이름을 입력 (창 제목에 포함된 이름과 일치)
3. 시/도 → 구/군/시 → 동 드롭다운에서 지역을 선택 후 **추가**
4. **시작** 클릭
5. 조건에 맞는 글이 올라오면 자동으로 응답 문구 전송

## 지원 작업명

- 싱막 (성막, 생막, 씽막 등 OCR 변형 포함)
- 하막
- 역류
- 누수
- 싱크대막힘
- 하수구막힘

## 수수료 조건

- 1~30%만 유효 (31% 이상 무시)
- `수수10`, `수수료 15%`, `수수 20` 등 다양한 표현 인식

## 주의사항

- 다크모드 미지원 (라이트 모드 사용)
- 절전/화면 잠금 시 동작 안 함
- 전송 시 카카오톡 창이 포커스되므로, 매크로 실행 중 다른 작업 시 주의
